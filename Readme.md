Display
=============

A PCD8544 (nokia 3110/5110) support for Java Graphics2D

It's possible:

* Write text
* Set the font
* Draw image
* Draw lines
* Draw rects
* Fill rects
* Draw circle
* (Maybe) all of [Graphics2D](https://docs.oracle.com/javase/7/docs/api/java/awt/Graphics2D.html) features (i'm not tested all)
